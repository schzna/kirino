# kirino

AES encryptor/decryptor.

This is for my study, I do not take responsibility for problems from using this.

## Supported Keysize

- [x] AES128
- [x] AES192
- [x] AES256

## Headers

- aes.hpp
in which AES objects are defined
- arith.hpp
implements functions of arithmetics in of Galois Field GF(2^8)
